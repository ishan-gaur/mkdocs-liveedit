# Edited Heading

This is a test site for the mkdocs-liveedit plugin.

## Features

- Click the edit button that appears upon hovering on a block
- Save with Cmd+Enter
- Changes rebuild automatically via livereload


## Try it out

Edit this paragraph by double-clicking on it. You should see a textarea
overlay appear with the raw markdown. Make your changes and hit Save.
